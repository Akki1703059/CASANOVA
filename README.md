# CASANOVA
A full-stack Airbnb-style web application developed by Aakash as part of an advanced backend project. CasaNova allows users to browse, create, review, and manage property listings with a clean UI and secure authentication.

What I Built

User signup & login using Passport.js
CRUD operations for Listings (add, edit, delete, view)
Users can write and delete Reviews
Flash messages for success/error
Proper error handling using custom middleware
Clean EJS pages with partials and layouts
MongoDB to store users, listings, and reviews

Stack Used

Node.js
Express.js
MongoDB + Mongoose
EJS (templating)
Passport.js for authentication
Bootstrap (basic UI)
